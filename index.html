About this site

This is a demonstration site for the csave-audio web application.

This application uses the Web Audio API to generate “CSAVE” sounds on the browser in real-time. It also uses the Service Worker API to cache files locally on the browser.

The source code is available from the SCM repository. Safari and other WebKit-based browsers are not fully supported, and it might ocassinally be broken from time to time.
